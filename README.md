# 📖 Contexto

Este repositório é dedicado ao estudo, documentação e desenvolvimento de Cyberdecks, computadores portáteis personalizados, projetados de acordo com as prioridades, o fluxo de trabalho e a criatividade de quem os constrói.

Inspirado na filosofia **DIY (Do It Yourself)** e na estética **High Tech, Low Life** do movimento Cyberpunk, este projeto busca unir conhecimento técnico, criatividade e liberdade na construção de um computador verdadeiramente personalizado.

---

# 🎯 Objetivos

- Estudar Raspberry Pi, ESP32, Arduino e outras plataformas embarcadas.
- Aprender eletrônica, Linux, impressão 3D e modelagem CAD.
- Desenvolver um Cyberdeck modular e expansível.
- Criar um laboratório portátil para estudos em segurança da informação, sempre em ambientes controlados e autorizados.
- Documentar todo o processo de pesquisa e construção.

## Resumos estruturados

### Conceito

Cyberdeck é um computador portátil personalizado inspirado na literatura cyberpunk, priorizando autonomia tecnológica, hardware aberto, cultura maker e software livre.

### Hardware

Principais tecnologias estudadas:

- Raspberry Pi
- LattePanda
- ESP32
- Arduino
- SDR
- GPIO
- Displays HDMI
- Impressão 3D

### Processo de construção

1. Pesquisa
2. Prototipagem
3. Modelagem CAD
4. Impressão 3D
5. Montagem
6. Configuração do sistema
7. Testes
8. Evolução contínua

## Glossário

| Conceito | Definição |
|----------|-----------|
| SBC | Computador em placa única. |
| GPIO | Interface de entrada e saída. |
| HAT | Placa de expansão para Raspberry Pi. |
| SDR | Rádio definido por software. |
| ESP32 | Microcontrolador com Wi-Fi e Bluetooth. |
| Arduino | Plataforma de prototipagem eletrônica. |
| Retrocart | Sistema modular de cartuchos. |
| DIY | Do It Yourself. |
